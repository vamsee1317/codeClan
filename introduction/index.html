JavaScript Implementation Methods

What is JavaScript?

JavaScript is a lightweight, interpreted scripting language used to make web pages interactive and dynamic. It runs directly inside the browser and allows developers to control page behavior.

  • Lightweight & easy to use
  • Interpreted language (runs line-by-line)
  • Object-oriented features
  • Event-driven programming
  • Supports both client-side & server-side
  • Cross-platform and widely supported

History of JavaScript

JavaScript was created by Brendan Eich in 1995 while working at Netscape. The language was built in just 10 days.

  1. Initially called Mocha
  2. Later renamed to LiveScript
  3. Finally became JavaScript

It was designed to make websites interactive and dynamic.

Ways to Add JavaScript to HTML

There are three main methods:

  • Inline JavaScript – JS inside HTML attributes
  • Internal JavaScript – JS inside <script> tag
  • External JavaScript – JS from a separate .js file

JavaScript Examples

<button onclick="alert('Button clicked!')">Click Me</button>
        
<script>
    function greet() {
        alert('Hello from Internal JavaScript');
    }
</script>

<button onclick="greet()">Internal JavaScript</button>
        
// app.js
function externalGreet() {
    alert('Hello from External JavaScript');
}

// HTML
<script src="app.js"></script>
<button onclick="externalGreet()">External JS</button>

Comparison of JS Methods

Feature Inline JS Internal JS External JS
File Separation ❌ No ⚠️ Partial ✔ Yes
Clean Code ❌ Poor ⚠️ Moderate ✔ Best
Reusability ❌ Low ⚠️ Medium ✔ High
